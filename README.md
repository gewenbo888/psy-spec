# Psy Spec · Psy 协议规范

A bilingual, deeply technical specification of Psy Protocol — realms, coordinators, state machine, runtime, networking, SDK, AI-native infrastructure, and the civilization-scale coordination layer that emerges from them.

The introductory specification: accurate enough for engineers to read with intent, abridged for clarity. Authoritative low-level documents (the proving-system constraints, the bincode schemas, the IR semantics) live in the source repositories; this site indexes them and explains the architecture they sit inside.

Single static HTML file. Runs entirely in the browser — no backend, no remote AI, no telemetry beyond aggregate visit counts. The "system modeler" is a local rule-based mapping of six dimensions (workload, realm count, prover capacity, latency, validator participation, agent share) to a stability outlook.
